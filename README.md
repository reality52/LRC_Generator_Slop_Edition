# Lyrics Generator Slop Edition


Десктопное приложение на Python для автоматического создания файлов субтитров (.lrc, .srt, .ass) из аудиофайлов с использованием нейросетей Whisper и Demucs.

## 🛠 Возможности

* **Разделение вокала**: Интеграция с Demucs (модель `htdemucs_ft`) для отделения голоса от музыки, что критически повышает точность распознавания в песнях.
* **Распознавание речи**: Использование OpenAI Whisper с выбором размера модели (от Tiny до Large) и поддержкой автоопределения языка.
* **Метаданные**: Автоматическое извлечение тегов (Артист, Название) из аудиофайлов через Mutagen и запись их в заголовок .lrc.
* **Встроенный плеер**: Предварительное прослушивание результата с синхронной подсветкой активной строки текста.
* **Гибкий экспорт**: Сохранение в форматы LRC (с поддержкой смещения/offset), SRT, ASS или обычный текст.
* **GPU Ускорение**: Поддержка CUDA для значительного ускорения обработки на картах NVIDIA.

## 📦 Зависимости и установка

Для работы требуются Python 3.8+ и установленный в системе **FFmpeg**.

1. **Базовые библиотеки**:
   ```bash
   pip install openai-whisper torch mutagen pygame
   ```

2. **Для разделения вокала (Demucs)**:
   ```bash
   pip install demucs
   ```

3. **Для работы на видеокартах NVIDIA (CUDA)**:
   Убедитесь, что установлена версия Torch с поддержкой CUDA. Проверить можно командой:
   ```bash
   python -c "import torch; print(torch.cuda.is_available())"
   ```

## 🚀 Краткая инструкция

1. Запустите скрипт: `python Qwen35Deepseeked.py`.
2. Нажмите **"..."** и выберите аудиофайл.
3. Если это песня, оставьте галочку на **Demucs** для очистки вокала.
4. Выберите модель Whisper (Small — оптимально для большинства задач).
5. Нажмите **СТАРТ**. Если модель не была скачана ранее, начнется её загрузка (от 150МБ до 3ГБ).
6. После появления текста проверьте синхронизацию в плеере и нажмите **СОХРАНИТЬ ФАЙЛ**.

## 📂 Пути кэшей по умолчанию

Приложение сохраняет временные файлы и веса нейросетей в следующие директории:

* **Модели Whisper**: `%USERPROFILE%\.cache\whisper` (Windows) или `~/.cache/whisper` (Linux).
* **Временный вокал**: В папке со скриптом создается директория `separated/`. Кнопка "📂 Вокал" открывает её автоматически.
* **Модели Demucs**: По умолчанию в `.cache/torch/hub/checkpoints`.

## 💻 Системные требования

* **Процессор**: Минимум 4 ядра.
* **Оперативная память**: 
    * 8 ГБ для моделей `tiny` / `base` / `small`.
    * 16 ГБ+ для моделей `medium` / `large`.
* **Видеокарта (опционально)**: NVIDIA с 4 ГБ+ VRAM для ускорения. Без неё используется CPU, что в 5-10 раз медленнее.
* **Место на диске**: ~10 ГБ (включая зависимости и закэшированные модели).

## ⚠️ Примечание
Кнопка **СТОП** прерывает выполнение процесса на любом этапе (будь то работа Demucs или Whisper). При очистке кэша моделей через интерфейс, при следующем запуске их придется скачивать заново.
